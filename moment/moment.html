Moment 套件介紹

Moment.js 官網

Moment.js 是一個強大的 JavaScript 套件，用於處理和操作日期與時間。它提供了方便的 API 來處理各種與時間有關的需求，如格式化、解析、相對時間顯示等。由於 JavaScript 本身對日期的處理並不是非常方便，Moment.js 極大地簡化了開發者的工作，並且在項目中處理日期與時間變得更加直觀和可靠。

這個套件特別適合需要進行複雜時間操作的情境，例如計算時間差、將時間轉換為特定格式，或者顯示相對於現在的時間（如 "5 分鐘前" 或 "明天" 這樣的描述性時間）。Moment.js 也支援多語言，因此無論是本地化的時間顯示還是處理不同時區的時間，這個套件都能得心應手地應對。

儘管 Moment.js 在早期非常流行，但由於其體積較大以及某些效能問題，現在官方也推薦考慮更輕量的替代方案，例如 date-fns 或 Day.js，這些替代方案提供了類似的功能但更加輕巧且模組化。不過，Moment.js 依然是一個非常成熟且功能強大的工具，許多現有的項目依舊在使用它。

範例程式碼


            <!-- Moment.js -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/moment.js/2.29.1/moment.min.js"></script>

<script>
    // 使用 Moment.js 處理日期
    const now = moment();
    console.log("現在的日期和時間：", now.format("YYYY-MM-DD HH:mm:ss"));
</script>
