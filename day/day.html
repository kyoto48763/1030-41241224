Day 套件介紹

Day.js 官網

Day.js 是一個輕量且強大的日期處理工具庫，專為那些需要簡單、快速操作日期的開發者而設計。它的 API 與 Moment.js 相似，但體積更小，效能更好，非常適合用於現代化的 Web 應用程式。Day.js 支持日期格式化、解析、相對時間計算等各種操作，並且以高度相容的方式滿足大多數應用場景。

範例程式碼


            <!-- Day.js -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/dayjs/1.11.7/dayjs.min.js"></script>

<script>
    // 使用 Day.js 處理日期
    const now = dayjs();
    console.log("現在的日期和時間：", now.format("YYYY-MM-DD HH:mm:ss"));
</script>
