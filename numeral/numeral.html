Numeral 套件介紹

Numeral.js 官網

Numeral.js 是一個輕量級的 JavaScript 庫，用於格式化和操作數字。它能夠幫助開發者輕鬆地處理貨幣、百分比、數字縮寫、字串解析等多種數字相關的任務，從而提高開發效率，減少代碼複雜性。Numeral.js 支援多種數字格式化樣式，例如千分位分隔符、貨幣符號、自定義格式等，使得展示數據時更加人性化和易讀。

範例程式碼


            <!-- Numeral.js -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/numeral.js/2.0.6/numeral.min.js"></script>

<script>
    // 使用 Numeral.js 處理數字格式
    const number = 10000;
    console.log("格式化後的數字：", numeral(number).format('0,0'));
</script>
