SweetAlert2 套件介紹

SweetAlert2 官網

SweetAlert2 是一個強大的 JavaScript 彈窗工具，用來替代傳統的 `alert()`、`confirm()` 等原生彈窗方法。SweetAlert2 具有高自定義性和美觀的外觀，使得彈窗效果更加現代化和友好，並且支持多種操作方式，非常適合用於 Web 開發中的各類提示、警告和確認框。

範例程式碼


            <!-- SweetAlert2 -->
<script src="https://cdn.jsdelivr.net/npm/sweetalert2@11"></script>

<script>
    // 使用 SweetAlert2 顯示彈窗
    Swal.fire({
        title: '成功!',
        text: '這是一個 SweetAlert2 的示例彈窗',
        icon: 'success',
        confirmButtonText: '確定'
    });
</script>
